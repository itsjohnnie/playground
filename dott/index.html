About

A single WebGL fragment shader runs the visual stages — source, preprocess, threshold, halftone, mask, color. Animated GIF frames are decoded on the CPU via the WebCodecs ImageDecoder API. Everything stays local.

Shortcuts
Space play / pause
S PNG snapshot
Drag and drop any file onto the canvas
Exports
PNG — current frame, raster, at canvas resolution
SVG — current frame, vector, transparent background
Loop — animated SVG cycling through every video / GIF frame
Inherit color — exports use currentColor so the SVG picks up its parent's CSS color
