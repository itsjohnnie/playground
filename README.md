## Claude Code Documentation

- **[Basic Version](https://itsjohnnie.github.io/playground/cc/basic.html)** — Clean documentation style

Topics covered:
- What is Claude Code
- Key features (tool use, agentic execution, context awareness)
- Claude vs Claude Code comparison
- Use cases and benefits
