Claude Terminal Embed Examples

Method 1: iframe Embed (Recommended for Webflow)

Use this URL pattern to embed in Webflow or any website:

<iframe
  src="https://your-domain.com/embed?s=YOUR_BASE64_SCRIPT&autoplay=true&controls=true"
  width="800"
  height="500"
  frameborder="0"
  allow="autoplay"
></iframe>

URL Parameters:

  • s - Base64 encoded script (required)
  • theme - "light" or "dark" (optional, defaults to system)
  • autoplay - "true" or "false" (default: true)
  • controls - "true" or "false" (default: true)
  • loop - "true" or "false" (default: false)

Method 2: Web Component (Advanced)

For custom implementations, use the web component directly:

<!-- Include the bundle -->
<link rel="stylesheet" href="/dist/embed/claude-terminal.css">
<script src="/dist/embed/claude-terminal.js"></script>

<!-- Use the component -->
<claude-terminal
  script="YOUR_BASE64_SCRIPT"
  theme="dark"
  autoplay="true"
  controls="true"
  loop="false"
></claude-terminal>

Getting the Script

To get the base64 encoded script:

  1. Create your animation at the main editor
  2. The URL hash contains your encoded script
  3. Copy everything after the # in the URL
  4. Use that as your s parameter or script attribute
