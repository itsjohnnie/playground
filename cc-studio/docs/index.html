Learn how to create beautiful animated Claude Code demos.

Getting started

Claudbon generates animated terminal demos that showcase Claude Code in action. Simply describe what you want to show, and AI will create a realistic animation script.

  1. Enter a prompt describing the demo you want to create
  2. AI generates a timeline of events (typing, tool calls, responses)
  3. Preview the animation in the terminal renderer
  4. Export as PNG or copy to clipboard

Animation events

user_input
User types a prompt at the > prompt with realistic typing animation
thinking_block
Shows Claude's internal reasoning with a shimmer effect
tool_call
Claude invokes a tool (Read, Edit, Write, Bash, Glob, Grep, Task, WebFetch, WebSearch, TodoWrite)
tool_result
Output from a tool, displayed with syntax highlighting
claude_response
Claude's streaming text response with markdown support
code_block
Standalone code block with diff formatting (+/- lines)
status_change
Changes the status line (accept edits, plan mode, diff stats)
pause
Adds a delay between events for readability

Status line states

The status line at the bottom of the terminal changes based on context:

base
? for shortcuts
accept_edits
>> accept edits on (shift+tab to cycle)
plan_mode
|| plan mode on (shift+tab to cycle)
background_task
1 background task · ? for shortcuts
diff
5 files +75 -61

Playback controls

Control your animation playback with these features:

  • Play/Pause — Toggle animation playback
  • Restart — Jump back to the beginning
  • Seek — Click anywhere on the progress bar to jump
  • Speed — Choose 0.5x, 1x, 1.5x, or 2x playback speed

Exporting

Once your animation is ready, you can export it in three ways:

  • Download PNG — Captures the current frame as an image
  • Download GIF — Exports the full animation as an animated GIF
  • Copy to Clipboard — Captures the image to paste anywhere

Easter egg

After an animation finishes playing, you can type in the terminal! Press Enter to send a message and get a response from Claude.

Stack

This project is built with:

  • Next.js 15 — App Router, React Server Components
  • TypeScript — Type-safe development
  • Tailwind CSS — Utility-first styling
  • pnpm — Monorepo with workspaces
  • Claude AI — Animation script generation
  • Phosphor Icons — Icon library
  • html-to-image — PNG export
